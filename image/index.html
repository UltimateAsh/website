Compétences en programmation

Langages de programmation de base :

Cette catégorie regroupe les langages de programmation fondamentaux servant de base à de nombreux autres.


Le langage C, reconnu pour sa syntaxe efficace et son accès bas niveau à la mémoire, est fréquemment employé dans le développement système et la programmation embarquée.


Le C++ étend les fonctionnalités du C en introduisant la programmation orientée objet.


Langages Web Frontend :

Cette catégorie se focalise sur les langages dédiés à la création de l'interface utilisateur et du contenu visuel des sites web.


Le HTML fournit la structure de base du contenu web, décrivant la hiérarchie des éléments.


Le CSS est utilisé pour styliser et mettre en page les pages web de manière esthétique.


Le JavaScript, un langage de programmation côté client, permet d'ajouter des fonctionnalités interactives et dynamiques aux sites web, améliorant ainsi l'expérience utilisateur.

Langages Web Backend :

Ces langages servent au développement de la logique serveur, à la gestion des bases de données et au traitement des requêtes des utilisateurs.


PHP, langage de script côté serveur, est largement utilisé pour créer des pages web dynamiques.


Symfony, un framework PHP, simplifie le développement en fournissant une structure organisée et des fonctionnalités réutilisables.


SQL est utilisé pour interagir avec les bases de données, facilitant le stockage, la récupération et la manipulation des données, souvent intégré dans le backend des applications web.
